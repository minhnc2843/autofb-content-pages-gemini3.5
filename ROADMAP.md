# Roadmap

## Phase 0 — Project Foundation

Tạo nền tảng tài liệu, memory, structure, rules để AI agent không phải đọc lại toàn bộ cuộc trò chuyện.

## Phase 1 — Core Laravel + Inertia React

App chạy local, có layout, routing, dashboard, topics, settings mock.

## Phase 2 — Pexels + Queue

Lấy media Pexels, tạo draft post, lưu queue.

## Phase 3 — Facebook Pages API

Đăng thật lên Facebook Page qua API chính thức.

## Phase 4 — Gemini AI

Dùng Gemini để viết caption, chấm điểm bài, phân tích media.

## Phase 5 — Page Insights + Page Audit

Đọc insight Page và tối ưu chiến lược nội dung.

## Phase 6 — Production Ready

Deploy ổn định, bảo mật token, scheduler, logs, backup.
