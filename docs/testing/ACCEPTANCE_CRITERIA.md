- App chạy local.
- Dashboard có layout.
- Topics CRUD hoạt động.
- Settings page hoạt động.
- Test pass.

- Pexels search hoạt động.

- Đăng được lên Page test.
- Lưu post id.
- Log lỗi rõ.

## Phase 4

- Gemini tạo caption.
- Gemini chấm điểm post.
- Lưu ai analysis.
