# Phase 1 — Core Laravel + Inertia React

## Mục tiêu

- Tạo Laravel app.
- Cài Inertia React.
- Cài Tailwind.
- Tạo layout dashboard.
- Tạo Topics CRUD.
- Tạo Settings page.
- Tạo Queue page mock.
- Tạo AI mock block.
- Chạy test nền tảng.

## Không làm trong phase này

- Không mở rộng scope ngoài danh sách mục tiêu.
- Không làm phần đăng thật nếu chưa đến Phase 3.
- Không gọi Gemini thật nếu chưa đến Phase 4.
- Không dùng browser automation.

## Acceptance Criteria

- [ ] App mở được tại http://127.0.0.1:8000.
- [ ] Dashboard hiển thị.
- [ ] Topics CRUD hoạt động.
- [ ] Settings lưu được dữ liệu mock.
- [ ] Queue page hiển thị dữ liệu mock.
- [ ] php artisan test pass.
- [ ] npm run build pass.

## Test cần có

- Unit test cho service chính của phase.
- Feature test cho route/controller chính.
- Manual test checklist được cập nhật trong `docs/testing/MANUAL_TEST_CHECKLIST.md`.
