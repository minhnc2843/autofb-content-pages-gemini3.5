# Product Brief

## Tên sản phẩm

Auto FB Content Planner

## Người dùng chính

Người quản lý nhiều Facebook Page muốn tìm media, tạo caption, xếp lịch bài đăng, duyệt nội dung và tối ưu Page bằng AI.

## Vấn đề cần giải quyết

- Mất thời gian tìm ảnh/video.
- Caption phải viết thủ công.
- Dễ đăng trùng nội dung.
- Khó theo dõi lịch đăng.
- Khó biết Page đang thiếu gì để tối ưu.
- Build AI agent dễ tốn token nếu không có memory rõ.

## Giải pháp

Một dashboard web app:

- Tìm media từ Pexels.
- Tạo caption theo preset/ngôn ngữ.
- Lưu queue bài viết.
- Duyệt bài trước khi đăng.
- Phase sau đăng thật qua Facebook Pages API.
- Phase sau dùng Gemini để chấm điểm và tối ưu Page.
