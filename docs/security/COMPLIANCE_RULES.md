# Compliance Rules

## Được làm

- Dùng Pexels API.
- Dùng Facebook Pages API.
- Dùng Gemini API.
- Lên lịch bài đăng Page nếu token/quyền hợp lệ.
- Tạo caption, queue, duyệt bài.

## Không được làm

- Tự động bấm trên giao diện facebook.com.
- Dùng browser automation để like/comment/share.
- Lách phát hiện nền tảng.
- Spam nội dung hàng loạt.
- Dùng media không rõ quyền.
