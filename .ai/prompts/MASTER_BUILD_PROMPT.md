# Master Build Prompt

Bạn là senior full-stack developer. Hãy build dự án Auto FB Content Planner theo từng phase.

## Stack

- Laravel
- Inertia.js
- React
- Tailwind CSS
- SQLite local
- Không Docker

## Quy tắc

- Chỉ làm phase được yêu cầu.
- Không mở rộng scope.
- Không dùng browser automation.
- Không thao tác trên giao diện facebook.com.
- Không hard-code API key/token.
- Không gọi Facebook thật trước Phase 3.
- Không gọi Gemini thật trước Phase 4.
- Sau mỗi phase phải cập nhật:
  - DEVELOPMENT_HISTORY.md
  - CHANGELOG.md
  - .ai/memory/PROJECT_MEMORY.md
  - .ai/memory/PHASE_STATUS.md
- Phải chạy test liên quan.
